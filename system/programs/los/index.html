LOS Language Reference

Variables

LOS uses dynamic typing. Variables are declared by assignment:

name = "John" age = 25 isActive = true numbers = [1, 2, 3, 4, 5]

Data Types

LOS supports: strings, numbers, booleans, arrays

text = "Hello" // String num = 42 // Number pi = 3.14159 // Float flag = true // Boolean items = [1, 2, 3] // Array

Operators

Math: + - * / %

Comparison: == != < > <= >=

Logical: and or not

String: + concatenates, str() converts to string

Print

print("Hello World") print("Value: " + str(variable)) print(42)

Conditionals

if age >= 18 { print("Adult") } else if age >= 13 { print("Teen") } else { print("Child") }

Loops

// While loop i = 1 while i <= 10 { print(i) i = i + 1 }

Functions

function add(a, b) { return a + b } result = add(5, 3) print(result)

Arrays

fruits = ["apple", "banana", "cherry"] print(fruits[0]) // First element print(len(fruits)) // Length fruits[] = "orange" // Add element

Built-in Functions

print(val) - Print value

len(arr) - Array length

str(num) - Convert to string

num(str) - Convert to number

rand(min, max) - Random number

input(prompt) - Get user input (prompt)

time() - Current timestamp

exit() - Exit program

cls() - Clear output
